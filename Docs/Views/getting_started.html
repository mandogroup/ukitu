Introduction

The Ukitu framework was designed to be as flexible as possible, its core purpose is to act as a KnockoutJS mobile friendly application platform and have the flexibility to sit within a complex MVVM approach but it can also largely automate certain processes to get a quick prototype or simple mobile site structure together with much less complexity.

This section is dedicated to a simple implementation of the framework to help you create a fairly basic site utilising the more common elements of the framework. For more advanced implementation techniques and knockoutjs integration see the Advanced and API Reference sections.
